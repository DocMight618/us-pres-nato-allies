# I will be back, will you spend? An Emotional Perspective on Alliance Burder Sharing

## Description of the project
What effects do emotions have on allies when it comes to collective security? While scholars who study emotions on international relations have linked emotions with alliance and international security, almost all of these scholars focus on 9/11 and Iraq, using qualitative analysis. With the development of large language models (LLMs), in this paper, I answer this question by quantifying emotions in U.S. presidential speeches from 1993 to 2023 using LLMs and examine the effects of six different emotions on NATO allies defense spending behavior. The results show that some emotions do matter more in the context of security alliance, and emotions have a strategic dimension, which can effectively influence the behavior of allies.
